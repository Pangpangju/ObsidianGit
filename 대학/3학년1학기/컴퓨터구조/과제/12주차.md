### EX3_3
Loop:
```
# iteration1
lw x7, 0(x10)
add x7, x7, x11
sw x7, 0(x10)

# iteration2
lw x8, 4(x10)
add x8, x8, x11
sw x8, 4(x10)

# iteration 2
lw x9, 8(x10)
add x9, x9, x11
sw x9, 8(x10)

addi x10, x10, 12
bne x10, x12, Loop
```
